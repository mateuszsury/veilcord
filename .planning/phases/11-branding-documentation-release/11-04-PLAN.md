<task type="auto">
  <name>Task 2: Create GitHub repository via gh CLI</name>
  <files>None (remote operation)</files>
  <action>
Create the GitHub repository using the gh CLI:

```bash
# Check if gh is authenticated
gh auth status

# If not authenticated, this will open browser for auth
gh auth login

# Create public repository
gh repo create mateuszsury/veilcord \
  --public \
  --description "Privacy-focused P2P encrypted messaging app with E2E encryption, voice/video calls, and file transfer" \
  --source=. \
  --remote=origin

# If repo already exists, just add remote
git remote add origin https://github.com/mateuszsury/veilcord.git 2>/dev/null || echo "Remote may already exist"
```
